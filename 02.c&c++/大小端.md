# 大小端

常见的笔试题：

判断当前的系统，是大端还是小端



## 小端

逻辑低位对应地址低位

对于整数1  对于的二进制：0000 0001

在内存地址中的顺序：1000 0000（地址由低到高）



## 大端

逻辑低位对应地址高位

对于整数1  对于的二进制：0000 0001

在内存地址中的顺序：0000 0001



判断大小端的方法：

```c
int x=1;
char *p=&x;
if(*p==1){
    printf("小端");
}else{
    printf("大端");    
}
```
